## 프로젝트 소개
교내 팀플의 팀원을 손쉽게 모집할 수 있는 사이트입니다.  


<br>

## 개발 인원 및 기간
- 개발 인원 : 백엔드 1명, 프론트엔드 2명
- 개발 기간: 2023.01~2023.02  
<br>

## 기능
- 게시판 
  -  CRUD 기능, 페이징, 카테고리별 필터링, 조회수, 북마크
- 댓글 
  - CRUD 기능, 대댓글/비밀댓글
- 마이페이지 
  - 작성한 글/북마크한 글/댓글 단 글 조회, 회원정보 변경
- 사용자
  - 회원가입 및 로그인(Security + Jwt), 유효성 검사  
  <br>
